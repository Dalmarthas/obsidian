# Question

What reusable motivation model does the current values-focused source add to the vault?

# Answer

The current source supports a five-part model:

1. Stop building effort on prestige hunger.
   `Be the best` is too comparative, unstable, and perfectionistic to work as a durable compass.

2. Separate values from impulses.
   Desire and emotion matter, but they swing too fast to carry long projects on their own.

3. Name the value behind the effort.
   Progress lasts longer when the person can say why the work matters beyond applause, status, or mood.

4. Use discomfort as a sorting test.
   In hard moments, ask whether comfort or progress deserves to win, and which choice tomorrow's self will respect more.

5. Apply early, not only after mastery.
   Immediate imperfect use of knowledge builds reinforcement loops that reduce avoidance and make learning more real.

The distinctive layer is that motivation is treated less as emotional intensity and more as values clarified under pressure.

# Comparisons

- Compared with [[wiki/syntheses/leadership/locus-of-control-motivation-framework|Locus of Control Motivation Framework]], this model is less about causal attribution and more about motivational quality. One asks whether action can matter; the other asks what makes action worth sustaining.
- Compared with [[wiki/concepts/leadership/meaning-through-responsibility|Meaning Through Responsibility]], this model is less duty-heavy and more self-interrogative: it begins with values clarification rather than with burden as the primary path to meaning.
- Compared with broader productivity advice, this framework is upstream of tactics. It explains why hacks, routines, and formulas fail when the person's motivational core is still confused or comparative.

# Gaps

- The branch currently rests on one secondary summary and needs stronger research depth.
- The source is strong on motivational diagnosis but weaker on how values are discovered, revised, or tested over time.
- The vault still needs a cleaner map of how values-driven action relates to intrinsic motivation, self-efficacy, and purpose-centered performance.
