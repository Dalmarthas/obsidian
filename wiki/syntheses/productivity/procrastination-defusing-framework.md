# Question

What reusable procrastination model does the current source base support beyond generic `just use discipline` advice?

# Answer

The current source base supports a six-part model:

1. Treat procrastination as a distorted choice field, not just weak character.
   The mind often sees only one dreaded ideal task and one attractive escape, while middle options vanish from awareness.

2. Reduce pain prediction before demanding output.
   Oakley is strongest here: procrastination often begins because the task already feels aversive before real work starts.

3. Replace `What should I do?` with `What can I do?`
   This shifts attention from shame-loaded ideals toward actions that are actually available.

4. Choose the next best meaningful step, not the tiniest symbolic one.
   Some `just start` advice fails because the proposed action is too trivial to justify the setup cost; a slightly larger unit can feel easier because it actually moves the work.

5. Use physical resets when the internal battle loops.
   Doorways, short walks, or room changes can interrupt a stuck working-memory conflict and make better choices visible again.

6. Distinguish real resets from disguised escape.
   Useful breaks support re-entry; procrastination extends avoidance. The difference is whether the reset reopens contact with the task or quietly replaces it.

The newer `mind-training` source adds a stronger anti-bargaining layer around the same problem: sometimes procrastination survives not because options are too narrow only, but because the person keeps taking the mind's negotiation process seriously. In those moments, doing one action the mind calls pointless, or allowing only `task or nothing`, may work better than further internal debate.

# Comparisons

- Compared with [[wiki/syntheses/productivity/kelly-mcgonigal-willpower-framework|Kelly McGonigal Willpower Framework]], this model is less about strengthening self-control capacity overall and more about redesigning the decision field inside a stuck moment.
- Compared with [[wiki/syntheses/productivity/barbara-oakley-learning-framework|Barbara Oakley Learning Framework]], this page is less learning-specific and more explicit about working-memory conflict, meaningful task sizing, and context-shift resets.
- Compared with [[wiki/concepts/productivity/dopamine-regulation|Dopamine Regulation]], this framework is less reward-biological and more choice-architectural. Dopamine regulation changes the temptation environment; this page changes what action becomes mentally available next.
- Compared with [[wiki/syntheses/productivity/mind-training-procrastination-framework|Mind-Training Procrastination Framework]], this page is more option-widening and task-design oriented, while the newer synthesis is more defusion-oriented and suspicious of the mind's entire bargaining game.

# Gaps

- The current evidence is still secondary-summary based rather than protocol-grade.
- The branch needs stronger evidence on when timed micro-starts outperform `next best thing` task sizing and when the reverse is true.
- The vault still needs more material distinguishing procrastination from burnout, executive dysfunction, depression, and legitimate incapacity.
- The newer source adds another unresolved question: when does `do the task or do nothing` produce clean action, and when does it become an overly rigid stance that ignores real recovery or nervous-system limits?
