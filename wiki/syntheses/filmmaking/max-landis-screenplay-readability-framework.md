# Question

What screenwriting model does the Max Landis source add to the vault?

# Answer

The source adds a six-part model:

1. Write for the read, not only the eventual movie.
   A screenplay should create pleasure and trust as a document, because that is how the industry usually meets it first.

2. Keep the page sparse and clear.
   Avoid prose bulk, needless direction, and any line that makes the script feel slower or more embarrassed than the movie it wants to become.

3. Make momentum visible everywhere.
   Every scene, line, and page should feel like it is carrying the story forward rather than sitting in place.

4. Treat structure as a tool.
   Three-act thinking is useful when it helps movement and clarity, but the writer should not mistake dogma for dramatic life.

5. Check character direction and spoken reality.
   Even without a classic arc, characters need a clear vector, and dialogue should survive read-aloud without sounding false.

6. Separate artistic craft from industry outcomes.
   Selling a spec, getting assignment work, and reaching production are connected but distinct arenas, so the writer should not confuse one kind of success with another.

# Comparisons

- Compared with Paul Thomas Anderson, the model is blunter and more commercially reader-facing; Anderson stresses blueprint discipline and actor room, while Landis stresses whether the read itself grabs the industry reader.
- Compared with Sorkin, the model is less about scene argument and rhetorical precision, and more about overall sparseness, pace, and avoiding dead weight on the page.
- Compared with the 2010 screenplay article, the model is less formal about structure and more explicit that three-act thinking is only useful insofar as it preserves momentum.
- Compared with the current `29 mistakes` diagnostic source, the model is more positive and energetic: it asks what makes a screenplay feel alive to read, not only what makes it fail.
