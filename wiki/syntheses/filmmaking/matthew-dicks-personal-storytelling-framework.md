# Question

What reusable personal-storytelling framework does the current Matthew Dicks source add to the vault?

# Answer

The source supports a six-part framework:

1. Find the five-second moment.
   Before shaping the story, identify the exact realization or irreversible internal shift. The useful test is whether the story can produce a sentence like `That was the moment I realized X`.

2. Build the before/after contrast.
   The opening state should be meaningfully different from the ending state. The audience needs to feel `I once thought/felt/was this; now I think/feel/am this`.

3. Start in motion.
   Begin with action rather than explanation. Drop the audience into a scene where something is already happening, then add only the context needed to make the moment legible.

4. Use misdirection to keep the audience active.
   State a plan that will fail, name an assumption that will be overturned, or plant breadcrumbs that invite prediction. The surprise should be unexpected but coherent.

5. Slow down before the reveal.
   Just before the realization, pause the forward rush. Add vivid concrete details and let anticipation build before naming the change.

6. Make the realization vulnerable.
   The point is not only to entertain but to reveal a true shift in fear, belief, pride, love, shame, capacity, or identity. That exposure is what turns the story into connection.

The compact rule is: personal stories work when suspense and specificity lead to a vulnerable moment of change.

# Basis

- [[wiki/sources/2026-04-21-storyworthy-by-matthew-dicks-core-message|STORYWORTHY by Matthew Dicks Core Message]] provides the core method: five-second moment, contrast, action, misdirection, delayed reveal, and vulnerability.
- [[wiki/concepts/filmmaking/transformation-in-storytelling|Transformation in Storytelling]] is the nearest canonical concept, but Dicks narrows transformation into a tellable personal realization.
- [[wiki/concepts/filmmaking/story-propulsion|Story Propulsion]] overlaps through delayed answers, failed plans, and audience prediction.
- [[wiki/concepts/communication/authentic-communication|Authentic Communication]] overlaps because the method depends on truthful vulnerability rather than polished anecdote alone.

# Comparisons

- Compared with [[wiki/syntheses/filmmaking/short-form-storytelling-framework|Short-Form Storytelling Framework]], Dicks is less platform-retention oriented and more change-moment oriented. Both care about openings, rhythm, and payoff, but Dicks starts from the internal realization rather than from visual hook and beat cadence.
- Compared with broader screenwriting pages, this framework is smaller and more portable. It does not require a full plot engine, antagonist, or act structure; it asks whether the speaker can dramatize one meaningful shift.
- Compared with [[wiki/concepts/communication/authentic-communication|Authentic Communication]], this gives authenticity a story shape: truthful vulnerability works better when it is structured through contrast, suspense, and specificity.

# Gaps

- The source does not fully cover Dicks's broader practice for finding story material over time.
- The model may be less useful for explanatory, technical, or collective stories where the main value is insight, evidence, or shared action rather than one person's realization.
- The vault still needs stronger cross-source comparison between personal storytelling, brand-origin storytelling, interview storytelling, and short-form creator storytelling.
