# Question

What reusable measurement framework does the Simon Sinek success-metrics source add to the vault?

# Answer

The current source base supports a six-part model:

1. Keep results, but refuse result-only scoring.
   Revenue, goals, and visible output still matter, but they are incomplete if used as the whole picture.

2. Add momentum as a real success variable.
   Ask whether capability, morale, demand quality, and healthy direction are strengthening over time rather than only whether the latest snapshot looks strong.

3. Measure trust because trust changes outcomes.
   Teams become more durable when scorecards notice who creates safety, reliability, and cohesion instead of rewarding only the loudest performers.

4. Evaluate the `how` alongside the `what`.
   A target hit through burnout, manipulation, or cultural damage should not count the same as a target hit through sustainable, values-aligned performance.

5. Distrust vanity wins.
   Rankings, labels, and short-term spikes can create prestige without proving lasting value.

6. Redesign incentives when scoreboards reward hidden damage.
   If the system keeps promoting low-trust stars or short-term spikes, the metric design is part of the problem rather than a neutral observer.

# Comparisons

- Compared with [[wiki/syntheses/leadership/simon-sinek-infinite-game-framework|Simon Sinek Infinite Game Framework]], this framework is more operational and scoreboard-focused. The infinite-game model says short-term metrics should not become the cause; this one says what a better measurement system should start noticing instead.
- Compared with [[wiki/syntheses/leadership/simon-sinek-trust-centered-leadership-framework|Simon Sinek Trust-Centered Leadership Framework]], this framework scales the trust logic upward from personnel selection into institutional metric design.
- The three Simon Sinek branches now complement each other:
  - the Golden Circle explains how purpose is communicated
  - the infinite-game model explains which time horizon should govern decisions
  - the success-metrics model explains how scoreboards and incentives should stop undermining that horizon

# Gaps

- This framework currently rests on one secondary summary rather than a deeper measurement-focused Sinek branch.
- The source is convincing on diagnosis but still light on precise implementation methods for momentum and trust measurement.
- The vault still needs stronger counterarguments from leaders who defend harder metric regimes and explain where softer indicators become too ambiguous.
