# What It Is

Zero-sum games are strategic situations where one player's gain is exactly another player's loss, so the total available value does not expand or contract. In the vault, the concept matters mainly as a warning about scope: some conflicts fit zero-sum logic, but many real business, negotiation, and coalition settings do not, because value can be created, destroyed, or recombined. Main source: [[wiki/sources/2026-04-13-what-actually-is-game-theory|What Actually Is Game Theory?]].

# Why It Matters

- It helps distinguish pure rivalry from situations where mutual gain is possible.
- It prevents the current strategy branch from treating every competitive field as a fixed-pie contest.
- It sharpens when aggressive win-lose logic is structurally accurate versus when it is strategically blinding.

# Evidence In This Vault

- The current source describes zero-sum games as the original strong focus of early game theory.
- It also argues that this frame is too narrow for many real settings:
  - players may create new value rather than only redistribute fixed value
  - coalitions may form
  - cooperation may improve total outcomes
- The source therefore treats zero-sum logic as one important subclass of game, not as the default shape of all strategic interaction.
- The newer repeated-games source adds a stronger practical warning against zero-sum reflex:
  - some strategies that `never lose` locally can still perform badly overall
  - tit for tat can win tournaments without beating opponents head-to-head because the real payoff comes from preserving high-value cooperation
  - many domains are better modeled as competing for value from the `banker` or environment than as taking value directly from the other player
- The newer Naval source adds a cultural and psychological version of the same warning:
  - status games are described as structurally comparative and difficult to finish
  - wealth creation is presented as more positive-sum because useful products and services can improve life for multiple parties at once
  - the page therefore now includes not only formal game structure but also everyday life domains where people mistakenly choose zero-sum scoreboards as their main game

# Source Comparisons

- Compared with [[wiki/concepts/business/game-theory|Game Theory]], this concept is narrower. Game theory covers strategic interaction broadly; zero-sum games describe only the fixed-pie subset.
- Compared with [[wiki/concepts/business/business-strategy|Business Strategy]], zero-sum logic is a diagnostic about the field, not a full plan for how to compete inside it.
- Compared with [[wiki/concepts/business/tit-for-tat|Tit for Tat]], zero-sum logic highlights where tit-for-tat reasoning is least appropriate. Tit for tat is strongest in repeated non-zero-sum settings where future cooperation can enlarge total payoff.
- Compared with [[wiki/concepts/business/mimetic-desire|Mimetic Desire]], zero-sum games are more structural while mimetic desire is more motivational: one explains the game shape, the other explains why people keep volunteering for bad games.

# Tensions / Open Questions

- Leaders often talk as if markets, organizations, or negotiations are zero-sum when they are only partly so.
- Declaring a field non-zero-sum can also become naive if real positional scarcity, rivalry, or coercion are ignored.
- The vault now has a cleaner repeated-games contrast, but it still needs stronger examples showing exactly when banker-style value creation is real and when it is wishful language layered over hard rivalry.
- The Naval source adds a neighboring risk: people may correctly reject status scoreboards in theory while still behaving as if social ranking were the real prize.
