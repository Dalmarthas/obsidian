# What It Is

Multivitamins and cognition is the claim that a standard daily multivitamin may improve older-adult cognitive performance and reduce apparent cognitive aging. In the current vault, this is a high-interest but lower-confidence concept because the effect is practical and surprising, yet currently rests on one compact secondary summary.

# Why It Matters

- It is one of the most actionable claims in the current brain-health source.
- It matters because it is unusually simple relative to many more exotic cognitive-aging interventions.
- It also gives the health domain a good test case for balancing evidence curiosity with skepticism.

# Evidence In This Vault

- The current source says large randomized trials found improved cognition, processing speed, and episodic memory in older adults taking a standard multivitamin.
- The same source frames the intervention as reducing effective cognitive aging by roughly `5` years.
- The branch treats this as easier to implement than many more demanding interventions.

# Source Comparisons

- Compared with [[wiki/concepts/health/biohack-skepticism|Biohack Skepticism]], this page sits in a productive tension: it is a supplement claim that sounds surprisingly strong, so it demands scrutiny rather than automatic dismissal or automatic belief.
- Compared with [[wiki/concepts/health/brain-health-and-cognitive-aging|Brain Health and Cognitive Aging]], this page is one specific intervention layer inside a larger prevention model built more confidently around exercise and sleep.

# Tensions / Open Questions

- This is one of the least settled-feeling claims in the branch and most clearly needs underlying paper review.
- The source does not fully explain whether the benefit reflects correcting deficiencies, general supplementation, or subgroup effects.
- The vault should treat this as promising but provisional rather than as established doctrine.
