# What It Is

Daily Dozen is the current branch's name for a repeatable checklist that turns plant-based nutrition from a general recommendation into a daily operating system. In this vault, the concept matters less as a perfect list of counted items and more as a practical bridge between evidence claims and everyday execution.

# Why It Matters

- It makes the nutrition branch actionable instead of merely ideological.
- It provides a repeatable structure for nutritional coverage and consistency.
- It keeps the vault's plant-based branch tied to behavior rather than to abstract agreement.

# Evidence In This Vault

- The Michael Greger source presents the `Daily Dozen` as a daily checklist for covering healthy foods and habits with enough range to support better outcomes.
- The same branch uses it as the main practical tool for implementing whole-food, plant-based eating.
- Exercise is also framed as a compatible daily habit layer rather than as a separate health silo.
- The source does not enumerate every checklist category in detail, so the page currently captures the operational role of the system more confidently than a fully itemized version of it.

# Source Comparisons

- Compared with [[wiki/concepts/health/whole-food-plant-based-diet|Whole-Food Plant-Based Diet]], this page is more operational and checklist-driven.
- Compared with [[wiki/concepts/health/lifestyle-medicine|Lifestyle Medicine]], this page is lower-level and more behavioral.
- Compared with [[wiki/concepts/health/healthspan|Healthspan]], this page describes a daily process, while healthspan names the long-term outcome that process is trying to improve.

# Tensions / Open Questions

- Checklists can clarify execution, but they can also become rigid or falsely complete if underlying purpose is forgotten.
- Because the current source only references the system at summary level, the vault still lacks a full primary breakdown of the exact checklist contents.
- A useful next question is which parts of the checklist carry the most benefit signal versus which parts mainly improve adherence and coverage.
