# What It Is

Supervised learning uses labeled examples to teach a model what a correct answer looks like, while unsupervised learning looks for structure or grouping in unlabeled data.

# Why It Matters

This distinction is one of the cleanest beginner entry points into machine learning because it clarifies how data and feedback shape what a model can learn.

# Evidence In This Vault

- [[wiki/sources/2026-04-09-google-beginner-ai-course]] uses supervised-versus-unsupervised learning as the main way to explain machine learning to beginners.
- The source stresses that supervised models compare outputs to known labels and adjust accordingly, while unsupervised models search for patterns without the same labeled correction loop.
- It also hints at semi-supervised extensions in the deep-learning layer.
