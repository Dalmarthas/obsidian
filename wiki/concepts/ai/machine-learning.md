# What It Is

Machine learning is the part of AI where models learn patterns from data in order to make predictions, classifications, or other useful outputs.

# Why It Matters

Machine learning is the first major narrowing of the AI umbrella into an operational training paradigm. It is where questions about data, labels, feedback, and model adjustment become central.

# Evidence In This Vault

- [[wiki/sources/2026-04-09-google-beginner-ai-course]] treats machine learning as the layer directly beneath AI and explains it primarily through data-driven training.
- The same source makes the supervised-versus-unsupervised distinction a core entry point for beginners.
- It also positions deep learning as a further specialization inside machine learning rather than a separate sibling category.
