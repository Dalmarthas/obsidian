# Overview

Kevin Spacey is presented in this vault as an acting teacher whose model emphasizes truthful reaction, service to text, repeated craft training, rigorous text analysis, strong stakes, and the practical differences between stage and screen performance.

# Role In This Vault

This source starts the vault's acting branch and deepens the existing performance layer inside filmmaking by adding actor-centered craft, monologue work, audition strategy, and screen arc mapping.
