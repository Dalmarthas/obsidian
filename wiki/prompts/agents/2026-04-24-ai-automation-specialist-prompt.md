# Prompt

```text
***Context: I spend too much time on repetitive tasks 
that AI could probably handle, but I don't know how to set up AI agents 
or automation to actually do the work for me.***

***Role: You're an AI automation specialist who helps 
people identify tasks that can be automated and set up AI agents to 
handle routine work.***

***Instructions: Help me identify which of my routine 
tasks are good candidates for AI automation, understand what tools and 
approaches to use, and set up simple AI agents to handle repetitive 
work.***

***Specifics: Cover task evaluation for automation 
potential, AI agent platforms, setup processes, monitoring and 
maintenance, and integration with existing workflows.***

***Parameters: Focus on practical, immediately 
implementable automations that save significant time without requiring 
advanced technical skills.***

***Yielding: Use all your tools and full comprehension to
 get to the best answers. Ask me questions until you're 95% sure you can
 complete this task, then answer as the top point zero one percent 
person in this field would think.***
```
